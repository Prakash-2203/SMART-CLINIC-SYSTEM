Issue title: User stories - Doctor
Body:

As a doctor, I can view the patient record so that patients have accurate medication.
As a doctor, I can see my scheduled appointments so that I can manage my day.
As a doctor, I can add prescription notes to an appointment so that pharmacy can fulfill medication.


Issue title: User stories - Patient
Body:

As a patient, I can view upcoming appointments so I can prepare for consultations.
As a patient, I can search for doctors by name or speciality so that I can book an appointment.
As a patient, I can cancel my booking so that I can manage changes in schedule.


Issue title: User stories - Admin
Body:

As an admin, I can add or remove doctors so that the clinic staff list remains up to date.
As an admin, I can view system reports of appointments so that I can track clinic utilization.
As an admin, I can assign roles and manage user accounts so that access is controlled.
